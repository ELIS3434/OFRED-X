## Requirements
- Python 3.10+
- pip or poetry

## Steps

1. **Clone repository**
```bash
git clone https://github.com/yourusername/AntiBot-Response-Manager.git
cd AntiBot-Response-Manager
```

2. **Install dependencies**
```bash
pip install -r requirements.txt
```

3. **Setup environment**
```bash
cp .env.example .env
# Edit .env with your API keys
```

4. **Run**
```bash
python src/main.py
```

## Docker

```bash
docker-compose up
```
